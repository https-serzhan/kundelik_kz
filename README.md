# Күнделік — Студенттік портал

Полноценное веб-приложение студенческого портала: Node + Express + SQLite, авторизация, личные данные, загрузка файлов, чат. Фронтенд — vanilla SPA (KZ/RU, тёмная/светлая тема).

## Запуск

Нужен Node.js 18+.

```bash
npm install      # установить зависимости (соберёт нативные модули better-sqlite3/bcrypt)
npm start        # запустить сервер
```

Открыть в браузере: http://localhost:3000

При первом старте база `kundelik.db` создаётся и наполняется демо-данными автоматически.

## Использование

1. Войти под администратором:
   - `admin@kundelik.local`
   - `admin12345`
2. В панели администратора создать студента: ФИО, email, пароль, группа, student ID, курс.
3. После этого студент входит только через обычную форму логина.

Публичная регистрация отключена: аккаунты студентов создаются только через панель администратора.

При необходимости данные администратора можно переопределить переменными окружения `ADMIN_EMAIL`, `ADMIN_PASSWORD`, `ADMIN_NAME`.

## Разделы

Главная (дашборд), Личный кабинет, Оценки (GPA), Расписание, Домашние задания, Посещаемость, Чат, Файлы, Экзамены. Тёмная/светлая тема, KZ/RU.

Для администратора открывается отдельная панель с собственным меню: обзор, создание студентов, выставление оценок по `student ID`, изменение персонального расписания, управление домашними заданиями и их оцениванием по каждому студенту, загрузка новых файлов, управление экзаменами, выставление посещаемости, административный чат, новости и объявления. Студенческие разделы вроде домашних заданий и GPA администратору напрямую не показываются.
